# 健康评估技能

## 触发条件
- 用户问进展、进度
- 用户要周报或月报
- 用户问"我最近怎么样"
- 每周自动触发汇总

## 执行流程

### 1. 收集数据
- 使用 `get_health_summary` 获取健康概要
- 使用 `get_diet_history` 获取饮食历史
- 读取 `/memories/goals.md` 获取目标进度

### 2. 模式分析
- 委派 pattern-detector 子代理分析长期趋势
- 识别营养缺口、饮食多样性、作息规律等

### 3. 生成报告
- 包含目标进度、营养趋势、行为模式、改进建议
- 对比上周/上月数据

### 4. 更新洞察
- 将分析结果写入 `/memories/insights.md`

## 输出格式
```
### 📊 健康评估报告

#### 目标进度
- 当前体重: Xkg → 目标: Xkg
- 进度: X% | 趋势: [on_track/off_track/stable]

#### 营养趋势（近7日均值）
| 营养素 | 均值 | 目标 | 达标率 |
|--------|------|------|--------|

#### 发现的模式
- [模式描述] (置信度: X%)
  → 建议: [具体建议]

#### 下周行动建议
1. [可执行的具体建议]
```
