# 营养分析技能

## 触发条件
- 用户上传食物图片
- 用户文字描述一餐内容
- 用户询问某食物的营养价值

## 执行流程

### 1. 加载用户画像
- 读取 `/memories/profile.md` 获取过敏原、疾病、饮食限制
- 读取 `/memories/goals.md` 获取当前目标和每日配额

### 2. 食物分析
- 使用 `analyze_food_image` 工具分析图片（如有图片）
- 或使用 `lookup_food_database` 查询文字描述的食物

### 3. 过敏原交叉检查
- 将分析出的食物成分与用户过敏原列表交叉比对
- 发现匹配时生成**严重程度分级的警告**

### 4. 目标影响计算
- 使用 `get_daily_status` 获取今日已摄入量
- 使用 `calculate_targets` 计算本餐占比和剩余配额

### 5. 生成个性化建议
- 结合过敏检查结果和目标影响
- 给出替代食物建议（按用户偏好排序）
- 给出份量调整建议

### 6. 更新记忆
- 使用 `learn_preference` 记录食物频次
- 更新 `/memories/nutrition.md` 的饮食摘要

## 输出格式
```
### 营养分析结果
| 项目 | 数值 |
|------|------|
| 总热量 | X kcal |
| 蛋白质 | Xg |
| 碳水 | Xg |
| 脂肪 | Xg |

### ⚠️ 健康警告（如有）
- [过敏原/疾病相关警告]

### 🎯 目标影响
- 本餐占今日配额的 X%
- 剩余配额：X kcal

### 💡 个性化建议
- [具体可行的建议]
```
